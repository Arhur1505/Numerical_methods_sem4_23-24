# Numerical Methods

This repository contains materials and tasks related to the "Numerical Methods" course offered in the fourth semester of Applied Computer Science at AGH University of Krakow.

## Course Topics

- Solving Linear Systems by Elimination Method
- LU Decomposition of Matrices
- Iterative Solution of Linear Systems
- Eigenvalues and Eigenvectors of Symmetric Matrices by Direct Methods
- Eigenvalues and Eigenvectors of Symmetric Matrices by Iterative Methods
- Determination of Single and Multiple Roots of Nonlinear Equations
- Polynomial Interpolation
- Interpolation using Splines
- Approximation
- Finding the Minimum Value of a Function
- Fast Fourier Transform
- Numerical Integration using Newton-Cotes Quadrature
- Numerical Integration using Gaussian Quadrature
- Pseudorandom Number Generators
- Monte Carlo Integration Method

All programs are written in C++ using the GSL library.
